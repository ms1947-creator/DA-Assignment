# Data Analytics Assignment
Overview

The assignment evaluates proficiency in SQL, Spreadsheet analysis, and Python programming.

Sections Covered

1. SQL – Hotel Management System
- Retrieved last booked room for each user  
- Calculated total billing amounts  
- Filtered high-value bills  
- Identified most & least ordered items monthly  
- Found customers with second highest billing  

2. SQL – Clinic Management System
- Revenue analysis by sales channel  
- Top 10 valuable customers  
- Monthly revenue, expense, and profit calculation  
- Most profitable clinics by city  
- Second least profitable clinics by state  

3. Spreadsheet Analysis
- Used INDEX-MATCH to map ticket creation time  
- Created helper columns for same-day and same-hour analysis  
- Used Pivot Tables for outlet-wise ticket insights  

4. Python Programming
- Converted minutes into human-readable format  
- Removed duplicate characters from string using loops  

Tools & Technologies used:
- SQL (MySQL)
- Microsoft Excel / Google Sheets
- Python
